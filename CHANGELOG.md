# Changelog

## 2026-04-17 — AI-Generated Event Descriptions

- Added `/api/openai/describe-event` endpoint using gpt-4o-mini to generate event descriptions.
- AI description always shown in event modal via a Google AI Overview-inspired card with shimmer loading skeleton.
- Ticketmaster description (`gig.info`) and `pleaseNote` shown below in secondary styling when available.
- Client-side cache by event ID to minimize API calls; max 2 retry attempts with fallback text on failure.
- Attribution label: "Description generated by OpenAI".
- Reusable `AiSummaryCard` component extracted for the AI card UI.
- Reusable `MapPinIcon` and `CalendarIcon` SVG icon components.
- Venue and date/time restyled as icon-labeled detail rows in the event modal.

## 2026-04-16 — Improve Suggestion Modal UI/UX

- Two-column grid of square, emoji-led mood buttons (single column on phone).
- Stronger contrast and hover lift against the modal background.
- Title restyled to the logo's Impact font, uppercase.
- Title and datepicker aligned with the grid bounds in a shared header row.
- Close button floats outside the top-right corner of the modal.
- Datepicker restyled with transparent background and divider between label and value.
